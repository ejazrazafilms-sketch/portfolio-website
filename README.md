# Ejaz Raza Films - Portfolio

A cinematic, interactive portfolio website for Syed Ejaz Raza Naqvi, a film director and documentary filmmaker.

## Features
- Custom interactive cursor
- Snap-scrolling vertical and horizontal sections
- High-quality cinematic image breaks
- Fully responsive across desktop, tablet, and mobile devices

## Development
This is a static site project using vanilla HTML, CSS, and JavaScript. A local development server is managed with Vite.

To run locally:
```bash
npm install
npm run dev
```

## Structure
- `index.html`: Main landing page featuring hero section, about, and press recognition.
- `work.html`: Portfolio gallery with a focus on director reels and on-set photography.
- `commercial.html`: Curated commercial TVC projects.
- `style.css`: All site styles, tokens, and responsive logic.
- `main.js`: Interactive logic for the cursor, loaders, and slider synchronization.
